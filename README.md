# Hexa Technik Website

Official website for **Hexa Technik - Industrial Automation Solutions**.
Built with React, Vite, Tailwind CSS, and Shadcn UI.

## Overview

Hexa Technik specializes in bringing top-tier industrial automation components to the Indian market. This website showcases our verticals, products, and provides a way for clients to reach out.

## Key Features

- **Dynamic Content**: Sections for Industry Verticals, Values, and Product Categories.
- **Product Showcase**: "Products & Brands" dropdown and page listing major partners (Panasonic, HIWIN, etc.).
- **Enquiry System**: 
  - Integrated Contact Form.
  - **WhatsApp Integration**: Direct message to support (+91 9225513779).
  - **Email Integration**: Auto-composed emails to hexatechnik@gmail.com.
- **Responsive Design**: Mobile-friendly navigation and layout.

## Project Structure

- `src/pages`: Main route components (Home, About, Products, Industries, Contact, etc.)
- `src/components`: Reusable UI components (Navbar, Footer, etc.)
- `src/index.css`: Global styles and Tailwind directives.

## Getting Started

1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Start the development server**:
   ```bash
   npm run dev
   ```

3. **Build for production**:
   ```bash
   npm run build
   ```
