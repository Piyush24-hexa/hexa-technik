# Luna Technologies Website

This is the official website for Luna Technologies Pvt. Ltd., built with React, Vite, and Tailwind CSS.

## Project Structure

- `src/pages`: Contains main route components (Home, About, Products, etc.)
- `src/components`: Reusable UI components
- `src/lib`: Utilities and helper functions

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```
